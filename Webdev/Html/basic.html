FOR LOOPS

For loops are used to iterate over a sequence (like a list, tuple, or string) or to repeat a block of code a certain number of times.


To use for loops you use the keyword for followed by a variable name, the in keyword, and the sequence you want to iterate over. The block of code to be executed is indented below.


EXAMPLE 1:

for(for i in range(5):
print("Hello World")

EXAMPLE 2:

for(for i in range(10):
print("The number is", i)

WHILE LOOPS

While loops are used to repeat a block of code as long as a specified condition is true.


To use while loops you use the keyword while followed by a condition. The block of code to be executed is indented below.


EXAMPLE 1:

while (x < 5):
print("Hello World")

EXAMPLE 2:

while (x < 10):
print("The number is", x)
x += 1

IF STATEMENTS

If statements are used to execute a block of code based on a condition.


To use if statements you use the keyword if followed by a condition. The block of code to be executed is indented below.


EXAMPLE 1:

if (x < 5):
print("Hello World")

EXAMPLE 2:

if (x < 10):
